# Laravel To-Do Application


This is a basic to-do application built by following a tutorial on [flynnsarmy website](https://www.flynsarmy.com/2015/02/creating-a-basic-todo-application-in-laravel-5-part-1/).


This to-do application has been extended to include a blog and admin functionality by following the Laravel 5 Beauty tutorial.
[L5 Beauty](http://laravelcoding.com/blog/laravel-5-beauty-sending-mail-and-using-queues?tag=L5+Beauty).
